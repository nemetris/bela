# error

Raises an error

## Syntax

```js
bela.error(text, [options])
```
Does not modify the subject.

## Usage

```js
bela.error('Page is not ready')
```

## Arguments

- text `string`

Text of the error to log.

- options `object`

An array of options.

## Examples

```js
bela.error('Page is not ready')
```
